Broad spectrum light source - > Band Pass filter ( blue ) -> now emits green photons -> emission filter ( band pass for green )

  • What happens to the light that is rejected by the filter?

    • Reflected back , appears still as "white light"

  • Band-Pass Filter = allows light of certain wave lengths only in the "band" to pass through

    • all other wavelengths longer or shorter are reflected

  • Long-Pass Filter = light that has wave lengths "longer" than the cut off point can pass through

    • all other wavelengths shorter are reflected

def band_pass_filter( fluorophore_data , midpoint_wavelength , bandwidth ):
    low_band = ( float( midpoint_wavelength ) - ( float( bandwidth ) / 2.0 ) )
    high_band = ( float( midpoint_wavelength ) + ( float( bandwidth ) / 2.0 ) )
    filtered_wavelengths = [ x for x in fluorophore_data[ "wavelength" ] if x >= low_band and x <= high_band ]
    filtered_wave_lengths_sum = sum( filtered_wavelengths )
    emission_sum = sum( fluorophore_data[ "emission" ] )
    percent_of_total_emission_spectrum_transmitted = ( ( filtered_wave_lengths_sum / emission_sum ) * 100.0 )
    return percent_of_total_emission_spectrum_transmitted


def long_pass_filter( fluorophore_data , low_bound_wavelength , hight_bound_wavelength=900 ):
    low_band = float( low_bound_wavelength )
    high_band = float( hight_bound_wavelength )
    filtered_wavelengths = [ x for x in fluorophore_data[ "wavelength" ] if x >= low_band and x <= high_band ]
    filtered_wave_lengths_sum = sum( filtered_wavelengths )
    emission_sum = sum( fluorophore_data[ "emission" ] )
    percent_of_total_emission_spectrum_transmitted = ( ( filtered_wave_lengths_sum / emission_sum ) * 100.0 )
    return percent_of_total_emission_spectrum_transmitted
