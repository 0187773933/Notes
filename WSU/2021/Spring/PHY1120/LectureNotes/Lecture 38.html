2 - Slit Interference

  • Superposition principle - 2 waves can propagate through each other , and at any point , they add
  • Destructive interference is a consequence of negative part of wave adding to a positive part
  • Geometry :

  • Y=L∗tan(θ) , but if θ is small ( means << L ) , then tan(θ)≈θ and y=L∗θ

    • θ is in radians!
    • Also , Δr=r2−r1

  • Remember :

    • If Δr=0,λ,2λ,3λ,etc=n∗λ

      • Then constructive interference
      • Crests add to crest
      • Bright

    • If Δr=λ2,3∗λ2,5∗λ2,etc=(n+12)∗λ

      • Then Destructive interference
      • Crests adds to trough
      • Dark

  • So if you know Δr to a point on the screen , you can say if bright or dark

  • Doing trigonometry , Δr≈α∗θ ( for small θ )

    • Eliminating θ gives : y=Δr∗Lα

     

  • Constructive Rule = y=n∗λ∗Lα

  • Destructive Rule = y=(n+12)∗λ∗Lα

  • Above equations are used to find location of bright and dark parts on the screen

 

Diffraction

  • Interference of 1 part of the slit with another part of the same slit

  • When top of slit and middle of slit have correct geometry for destructive interference , entire slit will destructively interfere with itself → produce dark region

  • From previous equation for destructive interference, but use d2 in place of α and 1st minimum at n = 0 :

    y=λ∗L2∗d2=λ∗L2

  • This is location on screen ( measured from mid point ) of the 1st "squash point" ( dark regions ) due to diffraction

  • Note : the "central bright region" in diffraction gets wider as d gets smaller

  • Note : if opening is very large ( think doorway or other macroscopic object )

    • Then the "bright region" will match up with ray prediction ( normal shadows )

      • You would only see diffraction effects at the edge of the doorway

    • And only if the edge of doorway sharp and light source is small and monochromatic ( single wavelength )
