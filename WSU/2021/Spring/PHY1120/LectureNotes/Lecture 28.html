Wave Speed

  • Two speeds:

    • Speed of mass in medium

      • The speed of a single point on a string, or of a single mass for linked beads
      • It's in the y-direction for transverse waves, so transverse to wave direction

    • Wave speed ( or wave velocity )

      • The "direction the crest of the wave is moving"
      • Here, this is the +x-direction

  • For strings, find that the wave speed is:

    v=Tension in the String (T)Mass Density of the String (μ)

    • μ = mass per unit length

      • NOT mass per unit volume

  • So v increases if T increases or if μ decreases, as seen in the activity

 

Superposition of Waves

  • For waves:

    • If f(x,t) is one wave, and g(x,t) is another

      • Then if they happen to occur at the same spot in a string, the result will just be the sum:
