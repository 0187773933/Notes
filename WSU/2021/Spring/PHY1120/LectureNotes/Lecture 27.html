  • Equation for wave

    • General Form = f(x−(v0∗t) or f(x+(v0∗t))

    • When you see the (x−(v0∗t)) or (x+(v0∗t)) as arguments of a function, it means some quantity (like height, pressure, etc.) will move left or right (in space) as time goes on

    • Example: cos(x−(v0∗t))

      • At t=0 , peak is at x=0

        • (x−(v0∗t))→(0−(v0∗t))=0
        • The argument of the cosine function is 0 , and cos(0)=1

  • Now try t=1

    • Where is the peak? Which peak are we talking about?
    • at t=0 , the picture looked like:

    • Cosine has many peaks

    • I'm following the peak that was at x=0 when t=0 , which is equivalent to the peak where the argument x−(v0∗t)=0

    • Ok, at t=1 , the argument becomes x−(v0∗1)

    • I want this to still equal 0 , so still get cos(0)=1

    • v0 is fixed, so x is something new

    • it will be the new location for some peak:

      • x−(v0∗1)=0→x=v0∗1

      • call this x1

        • x1=v0∗1

      • As t increases, x must increase

        • This means "our" peak is moving to the right

    • What happens if we have equation cos(x+(v0∗t))

      • Notice the ′+′ sign

      • As t increases:

        • x must decrease in order to keep the argument (x+(v0∗t)) the same value
        • So this wave is moving to the left

    • cos(x−(v0∗t)) means wave is moving to: +x

    • cos(x+(v0∗t)) means wave is moving to: −x
